## Level 1: eCitizen Login System
**Goal:** Implement secure access control using conditional logic.
* **Functionality:** Validates a hardcoded username (`adminKE`) and password (`254Secure`).
* **Logic:** Uses `if-else` statements and logical `AND` operators to ensure both credentials match before granting access.
* **Files:** `login.py`, `LoginSystem.java`, `login.cpp`

## Level 2: SACCO Member Financial Records
**Goal:** Manage and display structured data using Arrays and Loops.
* **Functionality:** Stores a list of 5 SACCO members and their respective savings balances.
* **Logic:** Utilizes **Parallel Arrays** (one for Strings, one for Integers) and a `for` loop to iterate through the data and print a formatted list.
* **Analytical Note:** Demonstrates an understanding of **Array Indexing**. An index out of bounds error would occur if the loop exceeded the array length (size 5).
* **Files:** `saccoo.py`, `SaccoSystem.java`
